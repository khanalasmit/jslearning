Learning Flexbox for Layouts

Posted on January 10, 2026

Flexbox is a powerful CSS layout tool that makes it easy to design flexible and responsive layouts. I used Flexbox to improve the navigation and blog list on my site. It helps align items, distribute space, and adapt to different screen sizes. If you are new to Flexbox, try experimenting with display: flex and its properties!

Flexbox Features I Used

  • Navigation bar with display: flex and gap
  • Blog list at the footer using Flexbox cards
  • Responsive design for mobile screens
